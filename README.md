# getOsakaCityVRSData
毎週初め頃更新される大阪市VRSデータを自身のGoogle Driveに保存するGASコード。<br>
スプレッドシートに追加し、トリガーとして水曜12時自動取得するようにしている<br>
スクレイピング用としてParsarライブラリver.8を使用している。<br><br>

該当ページ<br>
https://www.city.osaka.lg.jp/kenko/page/0000545560.html<br><br>
